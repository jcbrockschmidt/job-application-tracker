# 1.7 PDF Export

- [x] **`export:pdf` IPC** — use Electron's `webContents.printToPDF` to render the document; show OS save dialog via `dialog.showSaveDialog`; default filename: `<Full Name> - <Company Name> - Resume.pdf`; write file; return path
- [x] **Export error handling** — disk full, permissions error, path not found; each error shown inline with a specific message and "Save to different location" action (re-opens the save dialog)
