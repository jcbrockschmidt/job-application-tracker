# 1.6 Settings View

- [x] **Settings page** — accessible from sidebar footer icon and topbar icon; sections: API Key, Model, Contact Info, Theme, Backup Location
- [x] **API key field** — masked input; re-validate button; on success persist to keychain; inline error on failure
- [x] **Model picker** — dropdown populated from `settings:getAvailableModels`; saves selection to settings; used for all subsequent AI calls
- [x] **Contact info form** — pre-populated from stored settings; Save button calls `settings:save`
- [x] **Theme selector** — Light / Dark / System; applies immediately via MUI `ThemeProvider` and `useMediaQuery`
